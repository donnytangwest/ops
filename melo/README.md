# Melo 集中仓业务流程设计

本仓库包含 Melo 客户集中仓库区设计与作业流程展示页。

## 内容

- `melo/`：可直接发布的静态展示页面
- `fast_analyze_melo.py`：从本地业务数据生成 `melo/melo_analysis.json` 的分析脚本
- `inspect_workbooks.py`：Excel 字段结构探测脚本

## 数据口径

- 规划面积：截图完整区域为 12,000 平方米
- 出库数据：近30天订单
- 入库数据：近30天入库明细
- 库存数据：美西库存快照
- SOP参考：Melo 作业流 SOP 页面

原始 Excel 数据文件未提交到 GitHub。
